# wl-zhihu

### 项目介绍
使用vue3 + ts + bootstrap 构建的仿知乎项目

### 准备
掌握以下vue3新特性和ts语法，就可以进行愉快的开发了
#### vue3
+ setup函数，新的生命周期，computed和watch的用法
+ ref和reactive
+ Telepot和Suspense
+ hooks的写法
#### ts
+ 原始数据类型、any类型、数组
+ interface
+ 函数和类
+ 泛型
+ 类型别名，字面量和交叉类型
+ 声明文件和内置类型

#### 开发流程
+ 根据原型和设计图拆分公共组件，分析组件属性，定义好属性的接口，每个组件都要写使用文档

推荐使用vscode进行开发，安装ESLint和Vetur插件
